Projects

My interests lie in creating innovative solutions to complex problems. I enjoy exploring new technologies and methodologies to improve existing systems and develop new applications that can make a difference in people's lives.
Here are some of the projects I've worked on:


Project 1: GPT-2 IMDB fine-tuning + evaluation toolkit

Developed a toolkit to fine-tune GPT-2 on the IMDB dataset for sentiment analysis. The project involved data preprocessing, model training, and evaluation using various metrics to assess performance.

Project 2: Conversational RAG system (LangChain/LangGraph + FAISS + SentenceTransformers/Ollama)

Created a conversational retrieval-augmented generation (RAG) system using LangChain and LangGraph for building chatbots with contextual awareness. Integrated FAISS for efficient similarity search and SentenceTransformers/Ollama for embedding generation.

Project 3: Fraud / classification pipeline work (metrics, PR-AUC, cost-based thresholding)

Implemented a fraud detection classification pipeline that includes data ingestion, feature engineering, model training, and evaluation. Focused on metrics such as Precision-Recall AUC and cost-based thresholding to optimize model performance for real-world applications.

Project 4: Deepfake detection using ELA + CNN

Developed a deepfake detection system using Error Level Analysis (ELA) combined with Convolutional Neural Networks (CNN). The project involved image preprocessing, feature extraction, and training a CNN model to classify images as real or fake.

Project 5: Healthcare LLM evaluation framework (NIST/Stanford-aligned metrics)

Built an evaluation framework for large language models (LLMs) in the healthcare domain, aligning with NIST and Stanford metrics. The framework assesses model performance on tasks such as medical question answering, diagnosis prediction, and treatment recommendation.

Project 6: Portfolio website

Created a personal portfolio website using HTML, CSS, and JavaScript to showcase my projects, skills, and experiences. The website is responsive and optimized for various devices.
